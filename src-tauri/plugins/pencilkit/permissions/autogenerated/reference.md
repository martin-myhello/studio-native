## Default Permission

Default permissions for PencilKit plugin

#### This default permission set includes the following:

- `allow-is-available`
- `allow-show`
- `allow-hide`
- `allow-clear`
- `allow-get-drawing`
- `allow-set-drawing`
- `allow-get-image`
- `allow-set-tool`
